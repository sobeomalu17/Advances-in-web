Student Course Registration System

This project was developed to eliminate the inefficiencies of manual course registration by providing a simple and reliable online system.

Problem Solved

Manual registration processes were time-consuming and error-prone. This system automates course selection and registration.

Features

  • User authentication
  • Course selection
  • Admin management

Technologies Used

HTML, CSS, JavaScript

What I Learned

I learned how to structure multi-page websites, manage navigation, and design readable layouts for real users.
